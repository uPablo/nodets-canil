Projeto feito no módulo do curso Node + Typescript

### Pré-requisitos globais
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install`


### Para rodar o projeto
`npm run start-dev`

### Configurar o arquivo .env
`PORT=3000`
